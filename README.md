# Portfolio

Ryan Blakeman | rblakeman31@gmail.com | ryanblakeman.com

An interactive resume, a side-project I like to revisit over the years.

## Getting Started

```bash
yarn install
```

### Developing

```bash
yarn start
```

### Linting

```bash
yarn lint
```

### Typechecking

```bash
yarn typecheck
```

### Testing

```bash
yarn test
```

## Express Server

Hosts static information and images, currently just used for the "Events"

```bash
yarn server
```
